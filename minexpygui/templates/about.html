About This Project

MinexPy-GUI is a local Flask interface for MinexPy. It helps researchers run geochemical statistical workflows in the browser without writing Python scripts.

This release focuses on statistical analysis: upload data, preview rows, choose one or more element-concentration columns, and generate tabular summary metrics.

Learn more about MinexPy: GitHub Repository | Documentation
